# Nexus Dominion | Secure Gaming Hub

A high-performance, responsive gaming news and review portal built with **HTML5** and **CSS3**.

## Live Demo
[View the Live Secure Site Here](https://your-username.github.io/nexus-dominion/)

## Security Features
To meet modern web standards and rubric requirements, this project includes:
- **HTTPS Encryption**: Hosted via GitHub Pages with a secure SSL certificate.
- **Tab-Nabbing Protection**: External links use `rel="noopener noreferrer"`.
- **Input Validation**: The contact form uses regex patterns to prevent script injections.
- **Security Headers**: Includes a CSP meta tag to upgrade insecure requests.

## Responsive Design
The site utilizes a **CSS Flexbox** layout with media queries to ensure a seamless experience across:
- Desktop (Ultra-wide support)
- Tablets
- Mobile Devices (Vertical stack navigation)

## Project Structure
- `index.html`: The main landing page with featured news and secure external links.
- `reviews.html`: A secondary page featuring hardware reviews and a secure contact form.
- `styles.css`: The "Dark Mode" gaming aesthetic stylesheet.
- `README.md`: Documentation and project overview.
